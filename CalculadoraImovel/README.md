# Calculadora de Imóvel
Este é um programa em C# desenvolvido para calcular o valor de venda e aluguel de imóveis.

## Funcionalidades
*Calcula o valor do imóvel baseado no valor do m² da região.
*Define o valor do aluguel como 1% do valor da venda.
*Se for >4 quartos aumenta 10% o valor do imóvel.
*Calcula o aumento baseado no andar do prédio.

## Como rodar o projeto
1. Tenha o SDK do .NET instalado.
2. Clone o repositório
3. Execute o comando 'dotnet run'.
